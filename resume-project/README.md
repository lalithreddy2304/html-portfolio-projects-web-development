# HTML Resume Webpage

This project is a resume website created using pure HTML. It is designed to present personal and academic information in a clear and structured format without the use of CSS or JavaScript. The purpose of this project is to focus on mastering semantic HTML, page structure, and internal navigation.

---

## Project Overview

The HTML Resume Webpage includes the following key sections:

- *Profile/Introduction*
- *Education*
- *Skills*
- *Projects*
- *Contact Information*
- *Basic Contact Form*

The layout emphasizes proper use of semantic tags, sectioning, navigation links, and accessibility-friendly HTML practices.

---

## Learning Objectives

This project was created to:

- Practice semantic HTML tags (header, nav, main, section, footer)
- Implement internal navigation through anchor links
- Structure content in a clean, readable, and professional format
- Build a resume webpage using only HTML as a foundation before adding styling in future phases
