# HTML Projects

This repository contains a collection of HTML-only projects that I am developing as part of my learning and practice in web development. The primary objective of this repository is to build a strong foundation in HTML before progressing to CSS, JavaScript, and more advanced web technologies.

---

## Current Project

### 1. HTML Resume Webpage  
A structured resume webpage created using pure HTML.  
Features include:  
- Internal navigation through section-based links  
- Sections for Profile, Education, Skills, Projects, and Contact  
- Basic HTML form for contact

_File: resume.html (will be renamed to index.html once finalized)_
